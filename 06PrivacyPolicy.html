Why set up this website for？

I am a beginner in programming. I have been working in the civil engineering field for nearly 20 years. Because of my interest, I started to learn HTML, CSS, JavaScript... by myself, and started learning programming through books and the teaching channel on Youtube on February 12, 2021. In order to practice the results of self-study, decided to set up this blog. Through the process of realizing, I'll know whether I fully understand the meaning of programming grammar and how to use these programming grammars through the process of realizing. Then, my own ideas will be created through programming.


Most of the content of this blog took me about 2 months, but it was not complete, then cause of some reasons made me stagnate in learning programming and web page settings. Until the end of 2021, during the process of applying for the "WeHelp Software Engineer Support Program", let me understand that I really like programming and I have an ability to write programs. However, I should change the style of this blog because I no longer set up websites for job but like programming. As for the application of WeHelp, I was selected unexpectedly, but withdrew from the training camp due to physical factors. Perhaps it is a better way for me to set up this blog in a happy and comfortable state.
