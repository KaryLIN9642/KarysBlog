How to use excel to create a task gantt chart?

A Gantt chart is a one kinds of project management tool, designed by Henry Gantt around the years 1910~1915. A Gantt chart is a type of bar chart that illustrates a project schedule. In project management, the manager can monitor the progress of the current tasks of the project through the Gantt chart.

In the past, there were very few software that supported for for the production of Gantt charts. Now, there are many kinds of softwares that can draw Gantt charts.
